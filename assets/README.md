# 📁 **Assets Directory**

This directory contains all media, images, and documentation assets for the Richkeyrick Portfolio.

## 📂 **Directory Structure**

```
assets/
├── 📸 screenshots/          # Website screenshots for documentation
│   ├── hero-section.jpg
│   ├── profile-section.jpg
│   ├── portfolio-section.jpg
│   ├── ai-section.jpg
│   └── mobile-view.jpg
│
├── 🖼️ images/              # General images and graphics
│   ├── banner-portfolio.jpg
│   ├── logo.png
│   └── favicon/
│
├── 🎨 icons/               # Custom icons and SVGs
│   └── social/
│
└── 📄 docs/                # PDFs and documents
    ├── RESUME_RICHRICK.pdf
    └── PORTFOLIO_BROCHURE.pdf
```

## 📸 **Screenshots**

Screenshots untuk FEATURES.md dan dokumentasi.

**Dimensions:**
- Desktop: 1920x1080 atau 1440x900
- Mobile: 375x812 (iPhone X) atau 390x844 (iPhone 14)

**Format:** JPG (80-90% quality) atau WebP

## 🖼️ **Images**

**Banner/Header Image:**
- File: `banner-portfolio.jpg`
- Dimensions: 1920x600 atau 1920x400
- Format: JPG/PNG
- Max size: 500KB

**Logo:**
- Format: PNG (transparent)
- Sizes: 32x32, 64x64, 128x128, 512x512

## 🎨 **Icons**

Social media icons dan custom graphics dalam format SVG.

## 📄 **Documents**

Resume, portfolio brochure, dan dokumen lain dalam format PDF.

## 📝 **Notes**

- Semua assets akan di-host di CDN (ImageKit) untuk production
- Gunakan WebP format untuk better compression
- Optimize images sebelum commit
- Jangan commit video files (host di CDN atau YouTube)
