## 🎨 **COLOR PALETTE & BRAND GUIDELINES** {#colors}

### **Primary Colors**

```css
:root {
  /* Main Brand Colors */
  --primary-cyan: #00D4FF;      /* Neon Cyan - Main accent */
  --primary-gold: #FFD700;      /* Gold - Secondary accent */
  --primary-coral: #FF6B6B;     /* Coral - CTA/Highlight */
  
  /* Background Colors */
  --bg-dark-navy: #0f172a;      /* Main background */
  --bg-slate: #1e293b;          /* Card/Secondary background */
  --bg-card: rgba(30, 41, 59, 0.8);  /* Glassmorphism cards */
  
  /* Text Colors */
  --text-white: #ffffff;        /* Primary text */
  --text-gray: #94a3b8;         /* Secondary text */
  --text-muted: #64748b;        /* Muted text */
}
```

### **Color Usage Guide**

| Element | Color | Usage |
|---------|-------|-------|
| **Background** | #0f172a | Main page, sections |
| **Cards** | #1e293b or glass | Content containers |
| **Primary Accent** | #00D4FF | Buttons, links, highlights |
| **Secondary Accent** | #FFD700 | Stars, badges, achievements |
| **Danger/Warning** | #FF6B6B | Errors, important CTAs |
| **Success** | #00FF88 | Checkmarks, success states |
| **Text** | #ffffff | Headings, body text |
| **Muted Text** | #94a3b8 | Descriptions, labels |

### **Gradient Combinations**

```css
/* Hero Gradient */
background: linear-gradient(135deg, #0f172a 0%, #1e293b 50%, #0f172a 100%);

/* Accent Gradient */
background: linear-gradient(90deg, #00D4FF 0%, #FFD700 100%);

/* Card Glassmorphism */
background: rgba(30, 41, 59, 0.8);
backdrop-filter: blur(20px);
border: 1px solid rgba(255, 255, 255, 0.1);
```

## 💾 **FILE FORMAT & COMPRESSION** {#formats}

### **Format Selection Guide**

| Format | Best For | Transparency | Compression |
|--------|----------|--------------|-------------|
| **JPG** | Photographs, screenshots | No | High |
| **PNG** | Logos, icons, graphics | Yes | Medium |
| **WebP** | All web images | Yes | Highest |
| **SVG** | Icons, logos, illustrations | Yes | Vector |

### **Compression Settings**

**JPG:**
- Quality: 80-90%
- Progressive: Yes
- Optimization: Enabled

**PNG:**
- Compression: High
- Bit depth: 8-bit (for web)
- Transparency: Keep if needed

**WebP (Recommended):**
- Quality: 85%
- Method: 6 (slowest/highest compression)

### **File Size Targets**

| Image Type | Max Size | Optimal Size |
|------------|----------|--------------|
| Banner | 500 KB | 300-400 KB |
| Screenshot | 300 KB | 150-250 KB |
| Logo | 100 KB | 50-80 KB |
| Icon | 20 KB | 5-15 KB |

### **Compression Tools**

**Online:**
- [TinyJPG.com](https://tinyjpg.com) - JPG compression
- [TinyPNG.com](https://tinypng.com) - PNG compression
- [Squoosh.app](https://squoosh.app) - Advanced compression

**Software:**
- Adobe Photoshop (Export for Web)
- GIMP (Export optimized)
- ImageOptim (Mac)
- FileOptimizer (Windows)

## 🆘 **TROUBLESHOOTING**

### **Images Not Showing on GitHub**

**Problem:** Images appear as broken links

**Solutions:**
1. Check file paths are correct (case-sensitive)
2. Ensure files are committed and pushed
3. Wait 1-2 minutes for GitHub CDN cache
4. Try hard refresh (Ctrl+F5)

### **Images Too Large**

**Solution:**
1. Use TinyJPG.com to compress
2. Reduce quality to 80%
3. Consider WebP format

### **Wrong Dimensions**

**Solution:**
1. Use Canva to resize
2. Maintain aspect ratio
3. Crop if necessary
