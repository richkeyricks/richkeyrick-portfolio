---

## 🎯 **APA ITU PROJECT MOCKUP?**

**Bukan screenshot langsung dari website Anda**, tapi **screenshot dari interface produk AI Anda** yang dimasukkan ke dalam **device frame** (laptop/HP) di Canva.

**Contoh hasil:**
```
[Mockup Laptop/HP]
┌─────────────────────┐
│  📱 Device Frame     │
│  ┌───────────────┐  │
│  │ Screenshot    │  │  ← Interface HAINEO AI
│  │ Dashboard     │  │    (chat, analytics, dll)
│  └───────────────┘  │
└─────────────────────┘
    ↑ Professional presentation
```

## 🛠️ **2 CARA MEMBUAT MOCKUP:**

### **CARA A: Screenshot Interface Asli (Recommended)**

**Kalau produk AI Anda sudah ada/live:**

**Step 1: Screenshot Interface Produk**

| Proyek | Screenshot Dari | Fokus Pada |
|--------|----------------|------------|
| **HAINEO AI** | dashboard.haineo.ai atau app | Chat interface, AI response, sidebar menu |
| **HAI Platform** | app.hai-platform.com | Digital avatar, controls, settings |
| **NEO AI** | neo-ai dashboard | Search results, location data, analytics |
| **Creator AI+** | creator-ai dashboard | Image generation UI, prompt input, gallery |

**Step 2: Masukkan ke Canva Mockup**

1. Buka [Canva](https://canva.com)
2. Search template: **"laptop mockup"** atau **"phone mockup"**
3. Pilih template dark/tech style
4. Upload screenshot Anda
5. Drag screenshot ke dalam frame device
6. Export JPG/PNG

## 💡 **TIPS CANVA MOCKUP:**

**Template yang bagus di Canva:**
- Search: **"laptop mockup"** → pilih yang dark/black
- Search: **"phone mockup"** → pilih yang minimalis
- Search: **"tech mockup"** → untuk vibe cyberpunk

**Adjustments:**
- Background: Navy #0f172a atau dark gradient
- Device color: Hitam atau silver
- Screenshot brightness: Naikkan 10-20% agar terlihat jelas
- Add shadow: Subtle drop shadow untuk depth
