## 🛠️ **HOW TO CREATE THESE IMAGES**

### **Option 1: Canva (Recommended - Free)**

1. Go to canva.com
2. Create custom size: 1920 x 600 (for banner)
3. Search templates: "Technology banner", "Dark futuristic"
4. Customize colors: Dark navy + cyan accents
5. Add text: "RICHKEYRICK" etc.
6. Export as JPG

**Canva Template Keywords:**
- "Tech banner"
- "Futuristic header"
- "Dark gradient banner"
- "Neon tech"

---

### **Option 2: Screenshot Your Actual Website**

**For Screenshots:**
1. Open your portfolio website locally
2. Use browser dev tools (F12)
3. Enable device toolbar for mobile screenshots
4. Screenshot with: Windows + Shift + S (Win) or Cmd + Shift + 4 (Mac)
5. Save to assets/screenshots/

**Tools:**
- ShareX (Windows) - Advanced screenshots
- CleanShot X (Mac) - Professional screenshots
- Browser dev tools - Built-in
