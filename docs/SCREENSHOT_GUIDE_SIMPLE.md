**Bedanya:**
- Laptop = Lebar, banyak konten sekaligus
- HP = Sempit, konten ditumpuk ke bawah

**Cara ambil (2 cara):**

**Cara A: Pakai Chrome DevTools (Di Laptop)**
```
1. Buka richkeyrick.com di Chrome
2. Tekan tombol F12 (buka developer tools)
3. Klik icon HP (Toggle Device Toolbar)
   ↓
4. Pilih "iPhone 14 Pro" atau "Samsung Galaxy"
5. Website akan mengecil seperti di HP
6. Screenshot (Windows + Shift + S)
7. Save: mobile-view.jpg
```

**Cara B: Pakai HP Sungguhan**
```
1. Buka richkeyrick.com di HP Anda
2. Screenshot layar HP
3. Kirim ke laptop (WhatsApp/Email)
4. Save: mobile-view.jpg
```

**Yang harus kelihatan:**
- ✅ Website dalam ukuran layar HP (sempit)
- ✅ Navbar jadi menu hamburger (☰)
- ✅ Konten ditumpuk vertikal
- ✅ Masih terbaca dengan jelas

## 📸 **CARA SCREENSHOT DI WINDOWS:**

### **Metode 1: Snipping Tool (Paling Mudah)**
```
1. Tekan: Windows + Shift + S
2. Layar akan gelap, kursor jadi crosshair (+)
3. Klik dan drag untuk pilih area yang mau difoto
4. Foto tersimpan di clipboard (Ctrl + V untuk paste)
5. Buka Paint/Photoshop → Paste → Save
```

### **Metode 2: Tombol Print Screen**
```
1. Tekan: Print Screen (di keyboard, biasanya di atas)
2. Buka Paint
3. Tekan Ctrl + V (paste)
4. Crop (potong) yang tidak perlu
5. Save As → Pilih JPG → Save
```

### **Metode 3: ShareX (Gratis, Lebih Lengkap)**
```
Download: https://getsharex.com/
- Bisa screenshot area tertentu
- Bisa screenshot panjang (scrolling)
- Otomatis save ke folder
```
